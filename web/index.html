The translation is drawn right where you're reading.

Private by design

No cloud. No keys. No quotas.

Text recognition and translation run on-device with Apple's Vision and Translation frameworks. Nothing leaves your Mac.

✂️

Region capture

Drag to select any part of the screen. The translation appears in place, drawn over the blurred original.

🪟

Live overlay

A floating window you place over text — flip on Live mode to keep translating as the content changes.

🔒

Fully on-device

Powered by Apple's on-device models. Works offline, with no API keys and nothing sent to a server.
